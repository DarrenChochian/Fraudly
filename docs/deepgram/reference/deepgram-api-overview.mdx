## title: Deepgram API Overview

Deepgram's APIs allows you to interact with Deepgram programmatically.  You can use our APIs to:

* Build a [Voice Agent](/reference/voice-agent/voice-agent)

* Transcribe [streaming audio](/reference/speech-to-text/listen-streaming)

* Transcribe & analyze[ pre-recorded audio](/reference/speech-to-text/listen-pre-recorded)

* Transcribe [turn-based audio](/reference/speech-to-text/listen-flux)

* Transform a [continuous text stream](/reference/text-to-speech/speak-streaming)

* Transform a [single text request](/reference/text-to-speech/speak-request)

* Analyze [text](/reference/text-intelligence/analyze-text)

* Get all available [Deepgram models](/reference/manage/models/list)

* Get [model metadata](/reference/manage/models/get)

* Create [Temporary API Tokens](/reference/auth/tokens/grant)

* Administer your Deepgram account:
  * Manage [projects](/reference/manage/projects/list) and project [members](/reference/manage/members/list)
  * Manage project [invitations](/reference/manage/invites/list)
  * Manage user[ scopes](/reference/manage/members/scopes/list)
  * Get billing [balances](/reference/manage/billing/list)
  * Get billing [usage breakdown](/reference/manage/usage/breakdown/get)
  * Get project [requests](/reference/manage/requests/list)
  * Manage [API keys ](/reference/manage/keys/list)
  * Manage [self-hosted distribution credentials](/reference/self-hosted/distribution-credentials/list)

<br />

<Info>
  To create your first API key refer to our Guide [Creating API Keys](/docs/create-additional-api-keys).
